Image Annotator

Guidelines for annotating relations between images and captions

The following labels are available:

EVENT The image shows an event or situation described in the text
RESULT The image shows the result or aftermath of an event described in the text
PERSON The image shows a person described in the text
NON-PERSON The image shows a thing or animal described in the text
LOCATION The image shows the location mentioned (includes maps)
OTHER Unrelated, or doesn't fit into these categories
NOT-ENGLISH Text is not English

Select as many as apply, for example, an image that descirbes both a location and a person mentioned in the text should be annotated as both 'location' and 'person'.
